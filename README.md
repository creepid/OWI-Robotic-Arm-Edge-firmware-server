## OWI Robotic Arm Edge PCB and firmware control ##


### We need: ###

- AVR ATmega 8 (DIP16) (1)
- Hope RFM01 RF Receiver 434MHz (1)
- l293d motor drivers (3)
- L7805 (+5V, 1,5A, TO-220) (2)
- resistors, capacitors, some light-emitting diodes etc.
